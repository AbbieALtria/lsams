🚩 What is a Campaign? A batch of Lazada seller leads with a shared goal — e.g. "Ramadan Wave 1" or "Klang Valley Q3". Each campaign has its own status tracking and metrics.
♾️ Nothing is Ever Deleted Archiving a campaign keeps all data and history. Every priority change is logged with a reason and timestamp.
🔁 Same Seller, Two Campaigns A seller can appear in multiple campaigns — each tracks its own status independently.

How It Works — End to End
📂
Step 1
Upload CSV
Admin uploads seller list. Choose New Campaign (name + priority) or Add to Existing. Duplicate sellers within the same campaign are auto-skipped.
🚩
Step 2
Set Priority
On the Campaign Management page, set each campaign's priority number. P1 = worked first. Change any time — reason is saved to audit log.
📤
Step 3
Assign Leads
Leads land in the Pool. Supervisor assigns them to Gabay agents manually or via Smart Auto-Assign by territory.
📱
Step 4
Gabay Auto-Queue
Each agent's home screen shows P1 leads at the top automatically. No agent action needed — priority order is applied by the system.
📊
Step 5
Track Progress
Campaign Dashboard shows live metrics per campaign: total, unvisited, active, completed, and per-agent productivity table.
Priority Levels
P1 Highest Priority Gabay sees first
P2 High Priority After P1 done
P3 Normal Priority After P2 done
— No Priority Bottom of queue
