Scene 1
Introduction
Welcome — Supervisor & Admin Guide
Welcome, Admin! This guide covers everything you manage in LSAMS — from adding Gabay agents and uploading lead campaigns, to monitoring visits, correcting data entries, running AI health checks on your leads, and exporting reports for management.
Scene 2
Dashboard
Dashboard — Real-Time Command Center
Your dashboard shows the live state of the entire operation: total leads in the pipeline, how many sellers are now live, how many visits are logged today, and how many pool leads still need assignment. Gabay agent performance is ranked live — you can see who is hitting their target right now.
Total leadsLive sellersPool leads to assignAgent performance ranking
Scene 3
User Management
User Management — All System Users
Go to User Management to see every account in the system — Gabay agents, supervisors, and admins. Each row shows their role, WhatsApp mobile number (used by the bot), online status (🟢 green = active now), and assigned territory. Click Edit to update any user at any time.
All roles visible🟢 Online status dotWhatsApp numberLatest GPS per agent
Scene 4
User Management
Adding a New Gabay Agent
Click Add New User to create a Gabay agent account. Fill in their full name, choose a username they will use to log in, set a password, and assign their role as Gabay Agent. The WhatsApp number field is critical — this is how the bot identifies who is sending messages. Also set their territory so Smart Assignment routes the right leads to them.
Username + passwordWhatsApp number required for botAssigned city/territory
Scene 5
Lead Assignment
Assign Leads to Agents
The Assign Leads page shows all pool leads with no agent yet. You can manually select which agent gets each lead, or use Smart Auto-Assign which automatically distributes leads based on each agent's territory. Tick multiple leads and assign them in one click. Pool leads shown in orange on the dashboard need to be cleared daily.
Pool leads need clearingSmart Auto-Assign by territoryBulk selection
Scene 6
Campaign Upload
Import Leads — Choose Your Campaign First
Before uploading, choose the campaign. Click New Campaign to create one with a name and priority level — Priority 1 means Gabay agents will work on these leads first. Or click Add to Existing to append the new leads to a campaign already in the system. The same seller can appear in two campaigns — each tracks its own status independently. After upload, leads go to the pool ready for assignment.
New Campaign = name + priorityAdd to Existing = append onlySame seller allowed in 2 campaignsEach campaign = independent status
Scene 7
Visits Monitor
Visits — Monitor All Agent Activity
The Visits page shows every check-in from all agents in real time. You can see the time, agent name, seller visited, outcome selected, whether GPS was captured, how many photos were uploaded, and the agent's notes. Use the Filter button to narrow by agent, date range, or outcome. Export to Excel for reporting.
Real-time check-insGPS + photosFilter by agent/date/outcomeExport to Excel
Scene 8
Data Entry
Data Entry — Correct & Update Transactions
Data Entry allows you to edit visit records logged by Gabay agents. You can correct the outcome, update the follow-up date, add supervisor notes, or override the lead status. This is important when an agent logged the wrong outcome or missed adding notes. Every change you make is recorded in the Activity Log with your name and timestamp — full audit trail.
Edit outcomesUpdate follow-up datesOverride lead statusAudit trail — every change logged
Scene 9
Lead Health
Lead Health — AI Quality Check
Lead Health runs an AI analysis across your entire lead database and scores each seller from 0 to 100. It flags issues like duplicate addresses, missing contact numbers, warehouse addresses, and sellers already registered. Leads in red need immediate attention — fix them before sending agents to visit. Run this weekly to keep your data clean.
Duplicate addressesMissing contactsWarehouse flagsRun weekly
Scene 10
AI Scoring
AI Scoring — Priority Prediction
AI Scoring uses machine learning to rank which sellers are most likely to convert to live sellers. The model learns from your team's past visit history — which seller types, cities, and follow-up patterns lead to successful registrations. Leads with high scores should be visited first. Click Retrain Model monthly to keep the AI learning from new data.
ML modelHigh score = visit firstRetrain monthly

Reports
Reports — Team Performance Analytics
Reports gives you a full breakdown of team performance for any date range. Total visits, interested count, registrations, live sellers, and conversion rate per agent. Use Export Excel to send this to your manager or Lazada account team. You can filter by individual agent to review their month.
Any date rangePer-agent breakdownExport to Excel
Scene 12
Daily Digest
Daily Digest — End-of-Day Summary
The Daily Digest is your automatic end-of-day report — total visits today, interested sellers, registrations, and the top performing agent. It also alerts you to problems: agents with pending photo uploads, unassigned pool leads, and agents who did not log any visits today. You can email this digest directly to your manager.
Daily summaryInactive agent alertsPending photo alertsEmail to manager

Targets
Targets — Set & Track Monthly Goals
Use Targets to set monthly visit targets, live seller goals, and registration quotas for each agent. The progress bars update in real time as agents log visits. If an agent falls behind — shown in red — you know to follow up with them. Targets reset at the start of each month.
Per-agent targetsLive progress barsBehind = flagged redResets monthly

The Export page lets you download four key datasets: All Leads with their full status, Visit Logs with GPS and photo records, Registrations for the period, and Agent Performance summaries. These Excel files are ready to submit directly to Lazada or your company management without any manual formatting.

Activity Log
Activity Log — Full Audit Trail
The Activity Log records every action in the system — agent check-ins, data entry corrections you made, lead assignments, new user creation, and campaign uploads. If you ever need to verify what happened on a specific date or investigate a discrepancy, this is your source of truth. All entries show the user, timestamp, and exact change made.
Every action recordedWho + when + whatInvestigate discrepancies
Scene 16
Campaign Priority
Campaign Management — Set Who Works First
Go to Campaigns in the sidebar to manage all your campaigns. Each campaign has a Priority number — change it and click Save. A reason box will appear for the audit log. Priority 1 campaigns appear first in every Gabay agent's queue automatically — no manual action needed from agents. Archived campaigns stay in the system forever; their data and history are never deleted.
Priority 1 = Gabay sees these firstChange any timeArchive = data kept foreverReason saved to audit log

Campaign Dashboard
Campaign Dashboard — Live Metrics Per Campaign
The Campaign Dashboard shows you every campaign at a glance: total leads, how many are unvisited, in progress, completed, and closed. A progress bar shows completion percentage. Below each campaign is a Gabay team table — who has how many leads, how many they completed, and their completion rate. Use this to spot which agent or campaign needs attention.

Scene 18
WA Alert Control
WhatsApp Alerts — Mute or Enable Per Manager
As Super Admin you can control exactly who receives WhatsApp alert messages. On the User Management screen, each manager and supervisor with a mobile number has a green WA On button. Click it to mute that person — it turns grey and shows WA Off. Click again to re-enable. Alerts include: when a Gabay agent logs in, when a visit is logged, and when a seller status changes. Gabay agents are never affected by this toggle.
🔔 WA On = receiving alerts🔕 WA Off = mutedSuperadmin onlyLogin · Visit · Status change alerts
Scene 19
AI Lead Intelligence
AI Lead Intelligence — Auto-Scan on Import
Every time you import a campaign CSV, the system automatically scans each seller using Google Search (SerpAPI) + Claude AI in the background. No action needed. For existing campaigns, go to Campaign Dashboard → click ⚡ Run AI Sweep.
1Import CSV → AI enrichment queues automatically for every new lead
2Google Search finds seller across Shopee, TikTok, Facebook, Instagram, Tokopedia
3Claude Haiku extracts: category, price range, rating, followers, last active, Lazada status
4AI Score (0–100) saved — Gabay queue auto-sorts highest-score leads first within same campaign priority
5Campaign Dashboard shows: scanned count, avg score, high-potential count, already-on-Lazada count
Tip: For campaigns uploaded before this feature was added, go to Campaign Dashboard and click ⚡ Run AI Sweep on each campaign to backfill all scores.
